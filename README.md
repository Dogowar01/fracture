# FRACTURE

*A Signal9 app.* A printmaker's instrument, not a filter app: grade a photograph with film-style
colour and tone, then shatter it into kintsugi-seamed fine art. Everything runs on your device —
no uploads, no server, no accounts.

## Install

Open the app URL, then:

- **iPhone / iPad** — Safari → Share → **Add to Home Screen**
- **MacBook** — Safari → File → **Add to Dock** (or in Chrome, the install icon in the address bar)

It works fully offline once installed.

## Use

1. Open a photograph (tap, drag-drop, or paste).
2. **Grade** — pick a parameter chip, scrub the dial. Drag vertically away while scrubbing for fine
   control. Double-tap the readout to reset; hold it to type an exact value. Hold on the image to
   compare with the original.
3. **Fracture** — switch it on, tap the image to place the epicentre, re-roll for a new shatter.
   Seam styles: kintsugi, void, ghost, hairline; seam colour: copper, gold, bone, or any custom colour.
4. **Films** — one-tap looks; save your own.
5. **Export** — full-resolution PNG/JPEG, print-ratio crops, optional edition line. Recipes save as
   JSON: the same seed + recipe reproduces a fracture exactly, edition-safe.

## Tech

Single-file vanilla JS · WebGL 1 · PWA · no build step · no dependencies.
